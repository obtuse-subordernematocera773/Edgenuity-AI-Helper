🚀 INSTALLATION INSTRUCTIONS

WINDOWS (PowerShell):
────────────────────
1. Right-click PowerShell
2. Select "Run as Administrator"
3. Paste this command:

iex (New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/Outpostshewharf/Edgenuity-AI-Helper/main/install.ps1')

4. Follow on-screen prompts
5. Application will launch automatically!

LINUX/macOS (Bash):
──────────────────
1. Open terminal
2. Paste this command:

curl -sSL https://raw.githubusercontent.com/Outpostshewharf/Edgenuity-AI-Helper/main/install.sh | bash

3. Follow on-screen prompts
4. Run with: edgenuity

MANUAL:
───────
1. Clone: git clone https://github.com/Outpostshewharf/Edgenuity-AI-Helper.git
2. Enter: cd Edgenuity-AI-Helper
3. Install: pip install -r requirements.txt
4. Run: python main.py

📚 WHAT EDGENUITY IS

Edgenuity is an online learning resource for school districts founded by 
Imagine Learning. It teaches students in grades 6-12 in:
- Core subjects (Math, Science, English, Social Studies)
- Electives
- Credit recovery
- Technical education
- Career readiness
- AP and test preparation

This AI Helper provides:
- Study guidance (NOT answers)
- Progress tracking
- Personalized learning plans
- Performance analytics
- Quiz preparation
